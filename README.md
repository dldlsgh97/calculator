실생활 계산기
=============
### 제작목적
1.수학문제를 풀다가 계산기를 써야하는 상황에서 여러번 계산기를 쓰지않고 한번에 계산할 수 있는 계산기를 만들어보고 싶었다.

2.실생활에서 쓸 수 있는 간단한 기능을 가진 계산기를 만들어보고 싶었다.

### 사용 플랫폼
-구글 APP인벤터
